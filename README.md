# BitCommerz Admin Scaffold

This is a ready-to-run Vite + React scaffold (JSX) with Tailwind CSS and a componentized structure.
It includes the full module list extracted from your screenshots and a basic UI scaffold.

## Quick start

1. Install dependencies:
```bash
npm install
```

2. Run dev server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## To deploy on GitHub

- Create a new repo and push this project.
- Use GitHub Actions or Vercel/Netlify for CI/CD. Vite projects work well with Vercel.
